# git 使用简明教程

### 下载 git 二进制包

https://git-scm.com/downloads

选择对应os版本，默认安装即可

### 初始化配置

git config --global user.name xxx

git config --global user.email xxx@x.com

git config --list

### 提交代码

git init

git status
